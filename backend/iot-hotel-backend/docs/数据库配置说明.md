# 智慧酒店物联网控制系统 - 数据库配置说明

## 📋 数据库配置

本项目使用 MySQL 8.0 作为主数据库，存储所有酒店运营数据。

## 🔧 配置文件

### 1. 环境变量配置

文件位置: `backend/iot-hotel-backend/.env`

```env
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=iot_hotel_system
```

### 2. 数据库连接配置

文件位置: `backend/iot-hotel-backend/src/config/database.ts`

```typescript
import mysql, { Pool, PoolOptions } from 'mysql2/promise';
import config from '../config';

const poolOptions: PoolOptions = {
  host: config.database.host,
  port: config.database.port,
  user: config.database.user,
  password: config.database.password,
  database: config.database.database,
  waitForConnections: true,
  connectionLimit: 10,
  queueLimit: 0,
  charset: 'utf8mb4_unicode_ci'
};

const pool: Pool = mysql.createPool(poolOptions);

export default pool;
```

## 🚀 数据库初始化

### 1. 使用Docker初始化

```bash
cd docker
docker-compose up -d
```

数据库会自动执行 `docker/mysql/init/schema.sql` 中的初始化脚本。

### 2. 手动初始化

```bash
# 连接到MySQL
mysql -u root -p

# 执行初始化脚本
source /path/to/schema.sql
```

### 3. 使用MySQL客户端

1. 创建数据库: `iot_hotel_system`
2. 创建用户: `iot_user`
3. 执行 `docker/mysql/init/schema.sql` 中的SQL语句

## 🛠️ 常用SQL语句

### 1. 查询酒店信息

```sql
SELECT * FROM hotels LIMIT 1;
```

### 2. 查询房间列表

```sql
SELECT * FROM rooms 
WHERE room_status = 'available' 
  AND room_type = 'standard' 
ORDER BY id DESC 
LIMIT 10 OFFSET 0;
```

### 3. 查询预订列表

```sql
SELECT b.*, r.room_number, r.room_type 
FROM bookings b 
LEFT JOIN rooms r ON b.room_id = r.id 
WHERE b.status = 'pending' 
ORDER BY b.id DESC 
LIMIT 10 OFFSET 0;
```

### 4. 查询支付记录

```sql
SELECT * FROM payments 
WHERE status = 'paid' 
ORDER BY paid_at DESC 
LIMIT 10 OFFSET 0;
```

### 5. 查询会员列表

```sql
SELECT * FROM members 
WHERE member_level = 'gold' 
ORDER BY points DESC 
LIMIT 10 OFFSET 0;
```

### 6. 查询优惠券

```sql
SELECT * FROM coupons 
WHERE valid_from <= CURDATE() 
  AND valid_to >= CURDATE() 
ORDER BY id DESC 
LIMIT 10 OFFSET 0;
```

### 7. 查询送物订单

```sql
SELECT d.*, r.room_number 
FROM delivery_orders d 
LEFT JOIN rooms r ON d.room_id = r.id 
WHERE d.status = 'pending' 
ORDER BY d.id DESC 
LIMIT 10 OFFSET 0;
```

### 8. 查询报修工单

```sql
SELECT m.*, r.room_number 
FROM maintenance_tickets m 
LEFT JOIN rooms r ON m.room_id = r.id 
WHERE m.status = 'pending' 
ORDER BY m.id DESC 
LIMIT 10 OFFSET 0;
```

### 9. 查询服务评价

```sql
SELECT r.*, m.name as member_name, m.phone as member_phone 
FROM reviews r 
LEFT JOIN members m ON r.member_id = m.id 
ORDER BY r.id DESC 
LIMIT 10 OFFSET 0;
```

## 📊 数据库优化

### 1. 索引优化

```sql
-- 房间状态索引
CREATE INDEX idx_room_status ON rooms(room_status);

-- 预订状态索引
CREATE INDEX idx_booking_status ON bookings(status);

-- 支付状态索引
CREATE INDEX idx_payment_status ON payments(status);

-- 会员等级索引
CREATE INDEX idx_member_level ON members(member_level);

-- 优惠券有效期索引
CREATE INDEX idx_coupon_valid ON coupons(valid_from, valid_to);
```

### 2. 查询优化

```sql
-- 使用EXPLAIN分析查询
EXPLAIN SELECT * FROM bookings WHERE status = 'pending';

-- 使用覆盖索引
SELECT id, room_number, room_status FROM rooms WHERE room_status = 'available';
```
