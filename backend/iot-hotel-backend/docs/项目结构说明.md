# 智慧酒店物联网控制系统 - 后端服务

## 📚 项目说明

智慧酒店物联网控制系统后端服务，提供RESTful API、WebSocket、MQTT等服务。

## 🛠️ 技术栈

- **Node.js 20.x LTS**
- **Express.js 4.18+**
- **TypeScript 5.4+**
- **MySQL 8.0**
- **Redis 7.0**
- **MQTT Broker**

## 🚀 快速开始

### 1. 安装依赖

```bash
cd backend/iot-hotel-backend
npm install
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑.env文件，配置数据库、Redis、MQTT等连接信息
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
npm start
```

## 📡 API接口

启动服务器后，访问：

- **健康检查**: http://localhost:3000/api/v1/health
- **API文档**: http://localhost:3000/api/v1/docs

## 📊 数据库

### 创建数据库

```sql
CREATE DATABASE iot_hotel_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 执行SQL脚本

```bash
mysql -u root -p iot_hotel_system < sql/schema.sql
mysql -u root -p iot_hotel_system < sql/seed.sql
```

