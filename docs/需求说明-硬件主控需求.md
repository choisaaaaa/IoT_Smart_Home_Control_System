# 物联网智能家居控制系统 - 硬件主控需求

## 📋 文档概述

### 文档说明
本文档描述物联网智能家居控制系统的主控中心硬件需求，包括硬件组成、功能需求、接口定义、调试功能等。

### 设备定位
主控中心是系统的核心设备，负责指令发送、数据汇总、本地交互、AI语音助理等功能。

---

## 🔧 硬件组成

### 1. 核心控制模块

#### 1.1 ESP32主控板
- **型号**：ESP32-WROOM-32
- **参数**：
  - CPU：双核Xtensa LX6，240MHz
  - Flash：4MB
  - RAM：520KB
  - 无线：WiFi 802.11 b/g/n、蓝牙4.2 BLE
  - 接口：GPIO、UART、I2C、SPI、ADC、DAC
- **功能**：系统核心控制，数据处理，通信管理

#### 1.2 电源模块
- **参数**：DC 5V/2A
- **接口**：Micro USB
- **功能**：为主控和所有外设供电
- **稳压电路**：AMS1117-3.3V（3.3V稳压）

### 2. 人机交互模块

#### 2.1 显示模块
- **型号**：0.96寸OLED显示屏
- **分辨率**：128×64
- **接口**：I2C
- **功能**：显示系统信息、传感器数据、控制界面

#### 2.2 输入模块
- **按键模块**：4键独立按键
  - 按键1：模式切换
  - 按键2：灯光控制
  - 按键3：报警静音
  - 按键4：系统重置
- **麦克风模块**：INMP441（I2S数字麦克风）

#### 2.3 输出模块
- **扬声器模块**：3W 8Ω扬声器 + 功放芯片
- **蜂鸣器**：有源蜂鸣器，5V供电
- **状态指示灯**：RGB LED（WS2812B）

### 3. 连接材料
- **面包板**：400孔，带电源总线
- **杜邦线套装**：公对公、公对母、母对母，40根/套
- **排针排母**：2.54mm间距，40P
- **电源线**：DC 5.5×2.1mm接口

### 4. 外壳与结构
- **塑料外壳**：透明塑料盒，150×100×50mm
- **固定螺丝**：M3×10mm，4颗

## 🎯 功能需求

### 1. 网络通信管理

#### 1.1 云端通信
- **功能描述**：主控与云端服务器的通信管理
- **技术要求**：
  - 支持WiFi连接（802.11 b/g/n）
  - MQTT客户端功能（发布/订阅消息）
  - 断线自动重连机制
  - 心跳包保活（30秒间隔）
  - 数据加密传输（TLS 1.2）

#### 1.2 分控通信
- **功能描述**：主控与分控设备的内网通信
- **技术要求**：
  - WiFi AP模式或STA模式
  - MQTT Broker功能（简易版）
  - 设备发现与注册机制
  - 设备状态监控（心跳检测）
  - 消息队列管理

### 2. 指令调度与控制

#### 2.1 指令下发
- **功能描述**：接收云端指令并转发给对应分控
- **功能要求**：
  - 指令解析与路由
  - 指令优先级管理
  - 指令执行结果反馈
  - 超时重试机制（3次）
  - 指令日志记录

#### 2.2 数据上报
- **功能描述**：收集分控数据并上报云端
- **功能要求**：
  - 数据聚合与缓存
  - 数据压缩传输
  - 断网本地存储
  - 网络恢复后补传
  - 实时数据上报（1秒间隔）

### 3. AI语音助理

#### 3.1 语音识别
- **功能描述**：接收用户语音指令，识别意图
- **技术要求**：
  - 语音采集（麦克风模块）
  - 语音预处理（降噪、回声消除）
  - 语音识别（调用云端AI服务）
  - 意图识别（控制指令、查询指令）
  - 识别准确率＞90%

#### 3.2 语音反馈
- **功能描述**：语音播报系统状态和执行结果
- **技术要求**：
  - 语音合成（TTS）
  - 扬声器播放（3W 8Ω）
  - 音量调节（0-100%）
  - 语音库（中文）
  - 播报内容：温度、湿度、设备状态、报警信息

#### 3.3 语音指令集
- **控制指令**：
  - "打开客厅灯" → 控制分控1的RGB LED
  - "关闭插座" → 控制分控1的继电器
  - "打开门锁" → 控制分控2的电控锁
  - "设置温度26度" → 发送温度设置指令
- **查询指令**：
  - "现在温度多少" → 播报当前温度
  - "空气质量怎么样" → 播报空气质量
  - "门锁状态" → 播报门锁状态
- **情景指令**：
  - "回家模式" → 执行回家情景
  - "离家模式" → 执行离家情景
  - "睡眠模式" → 执行睡眠情景

### 4. 本地人机交互

#### 4.1 按键控制
- **功能描述**：通过物理按键控制设备
- **按键定义**：
  - **按键1**：模式切换（环境模式/安防模式/设置模式）
  - **按键2**：灯光控制（开/关/调亮/调暗）
  - **按键3**：报警静音（停止蜂鸣器报警）
  - **按键4**：系统重置（恢复出厂设置）
- **交互反馈**：按键按下时LED闪烁确认

#### 4.2 显示屏显示
- **功能描述**：OLED显示屏显示系统信息
- **显示内容**：
  - **主界面**：当前模式、时间、温度、湿度
  - **环境界面**：温湿度、空气质量、光照强度
  - **安防界面**：门锁状态、门窗状态、报警信息
  - **设置界面**：WiFi设置、服务器设置、系统信息
- **显示切换**：按键1切换不同界面

#### 4.3 状态指示
- **功能描述**：LED指示灯显示系统状态
- **指示灯定义**：
  - **红色LED**：系统错误或报警（闪烁）
  - **绿色LED**：系统正常运行（常亮）
  - **蓝色LED**：WiFi连接中（呼吸灯）
  - **黄色LED**：分控离线（闪烁）

### 5. 网络参数配置

#### 5.1 WiFi配置
- **功能描述**：在线修改WiFi连接参数，无需重新烧录
- **配置方式**：
  - **方式1**：Web配置界面（主控提供AP热点）
  - **方式2**：手机App配置（蓝牙或WiFi直连）
  - **方式3**：按键+显示屏配置（离线配置）
- **配置项**：
  - WiFi SSID
  - WiFi密码
  - IP地址模式（DHCP/静态）
  - 静态IP地址、网关、DNS

#### 5.2 服务器配置
- **功能描述**：在线修改云端服务器连接参数
- **配置项**：
  - MQTT服务器地址
  - MQTT服务器端口
  - MQTT用户名
  - MQTT密码
  - WebSocket服务器地址
  - 数据上报间隔

#### 5.3 配置存储
- **功能描述**：配置参数持久化存储
- **存储方式**：ESP32 Flash存储（Preferences库）
- **存储内容**：
  - WiFi配置
  - 服务器配置
  - 设备ID
  - 系统配置
- **恢复默认**：按键4长按3秒恢复出厂设置

## 🔌 接口定义

### 1. GPIO引脚分配

#### 1.1 显示屏接口（I2C）
- **SDA**：GPIO 21
- **SCL**：GPIO 22
- **VCC**：3.3V
- **GND**：GND

#### 1.2 按键接口
- **按键1**：GPIO 16（模式切换）
- **按键2**：GPIO 17（灯光控制）
- **按键3**：GPIO 18（报警静音）
- **按键4**：GPIO 19（系统重置）
- **上拉电阻**：10KΩ（内部上拉）

#### 1.3 麦克风接口（I2S）
- **SCK**：GPIO 26
- **WS**：GPIO 25
- **SD**：GPIO 33
- **VCC**：3.3V
- **GND**：GND

#### 1.4 扬声器接口
- **正极**：GPIO 25（PWM输出）
- **负极**：GND

#### 1.5 蜂鸣器接口
- **正极**：GPIO 27
- **负极**：GND

#### 1.6 RGB LED接口（WS2812B）
- **数据**：GPIO 2
- **VCC**：5V
- **GND**：GND

### 2. 通信接口

#### 2.1 WiFi接口
- **天线**：板载天线
- **频段**：2.4GHz
- **协议**：802.11 b/g/n

#### 2.2 蓝牙接口
- **版本**：BLE 4.2
- **天线**：板载天线

#### 2.3 USB接口
- **类型**：Micro USB
- **功能**：供电 + 串口通信
- **波特率**：115200
