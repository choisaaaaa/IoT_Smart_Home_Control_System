### 1.2 架构分层

#### 1.2.1 表现层（Presentation Layer）
- **Web管理后台**：Vue.js 3 + Ant Design Vue
- **手机端H5/小程序**：Vue.js PWA / 小程序
- **前台电脑GUI**：Electron + Vue.js
- **ESP32终端界面**：轻量级Web服务器

#### 1.2.2 业务逻辑层（Business Logic Layer）
- **后端服务**：Node.js + Express
- **AI服务**：智谱GLM-4-Flash
- **MQTT服务**：Mosquitto Broker
- **WebSocket服务**：Socket.io

#### 1.2.3 数据访问层（Data Access Layer）
- **主数据库**：MySQL 8.0
- **缓存数据库**：Redis 7.0
- **消息队列**：MQTT Broker

#### 1.2.4 设备层（Device Layer）
- **前台管理端**：ESP32-S2-MINI / STM32F103C8T6
- **楼控**：ESP32-WROOM-32
- **客房端**：ESP32-WROOM-32

### 5.2 认证授权

#### 5.2.1 用户认证
- **JWT Token**：登录后颁发Token，有效期24小时
- **Refresh Token**：用于刷新Token，有效期7天
- **登录方式**：用户名密码、手机验证码

#### 5.2.2 设备认证
- **设备ID + 设备密钥**：设备启动时认证
- **Token机制**：认证通过后颁发设备Token
- **心跳检测**：30秒心跳，异常离线报警

#### 5.2.3 权限管理
- **RBAC（基于角色的访问控制）**
- **角色定义**：
  - 管理员：所有权限
  - 前台员工：入住/退房、设备控制
  - 服务员：客房服务、报修处理
  - 住客：个人设备控制、服务请求

## 📝 选型总结

### 8.1 选型原则

1. **成熟稳定**：选择经过市场验证的技术
2. **生态完善**：拥有活跃的社区和丰富的文档
3. **性能优秀**：满足系统性能要求
4. **易于维护**：代码可读性高，维护成本低
5. **成本可控**：开源技术为主，降低授权成本
6. **扩展性强**：支持水平和垂直扩展

### 8.2 技术栈优势

| 技术栈 | 优势 |
|--------|------|
| Vue.js 3 | 轻量级、高性能、生态完善 |
| Node.js | 事件驱动、非阻塞I/O、适合实时应用 |
| MySQL | 关系型数据库、事务支持、成熟稳定 |
| Redis | 内存数据库、高性能、支持多种数据结构 |
| MQTT | 轻量级、低带宽、适合物联网 |
| ESP32 | 低成本、WiFi+蓝牙、双核性能 |
