## 🎯 快速开始

### 方案1: 本地开发（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/your-username/iot-smart-hotel.git
cd iot-smart-hotel

# 2. 启动开发环境
docker-compose up -d

# 3. 本地开发
cd backend
npm install
npm run dev
```

**参考文档**: [跨平台开发环境配置指南](./跨平台开发环境配置指南.md)

---

### 方案2: 云服务器部署（比赛演示）

```bash
# 1. 在云服务器上执行
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER

# 2. 克隆项目
git clone https://github.com/your-username/iot-smart-hotel.git
cd iot-smart-hotel

# 3. 部署
docker-compose up -d
```

**参考文档**: [Ubuntu云服务器部署指南](./Ubuntu云服务器部署指南.md)

## 🐛 常见问题

### Q1: Windows和Ubuntu环境不一致怎么办？
**A**: 使用Docker容器化开发，确保环境完全一致。参考[跨平台开发环境配置指南](./跨平台开发环境配置指南.md)

### Q2: 本地开发如何连接云服务器数据库？
**A**: 修改`.env`文件中的数据库地址为云服务器IP。参考[Docker部署和访问指南](./Docker部署和访问指南.md)

### Q3: 如何实现自动化部署？
**A**: 使用GitHub Actions + SSH实现CI/CD。参考[Ubuntu云服务器部署指南](./Ubuntu云服务器部署指南.md)
