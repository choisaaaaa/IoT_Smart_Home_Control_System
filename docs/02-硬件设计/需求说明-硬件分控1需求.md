# 物联网智能家居控制系统 - 硬件分控1需求

## 📋 文档概述

### 文档说明
本文档描述物联网智能家居控制系统的分控1（环境监测与设备控制）硬件需求，包括硬件组成、功能需求、接口定义、调试功能等。

### 设备定位
分控1负责环境数据采集和家居设备控制，包括温湿度、空气质量、光照监测，以及继电器控制、RGB LED控制、红外遥控学习等功能。

---

## 🔧 硬件组成

### 1. 核心控制模块

#### 1.1 ESP32分控板
- **型号**：ESP32-WROOM-32
- **参数**：
  - CPU：双核Xtensa LX6，240MHz
  - Flash：4MB
  - RAM：520KB
  - 无线：WiFi 802.11 b/g/n、蓝牙4.2 BLE
  - 接口：GPIO、UART、I2C、SPI、ADC、DAC
- **功能**：环境监测、设备控制、通信管理

#### 1.2 电源模块
- **参数**：DC 5V/1A
- **接口**：Micro USB
- **功能**：为分控和所有传感器供电
- **稳压电路**：AMS1117-3.3V（3.3V稳压）

### 2. 环境监测传感器

#### 2.1 温湿度传感器
- **型号**：DHT11
- **参数**：
  - 温度范围：0-50℃
  - 湿度范围：20-90%RH
  - 精度：温度±2℃，湿度±5%RH
- **接口**：单总线数字接口

#### 2.2 空气质量传感器
- **型号**：MQ-135
- **参数**：
  - 检测气体：CO、酒精、苯、氨气等
  - 输出：模拟电压信号
  - 工作电压：5V
- **接口**：模拟输出

#### 2.3 光照传感器
- **型号**：光敏电阻模块
- **参数**：
  - 电阻范围：10KΩ-1MΩ
  - 输出：模拟电压信号
- **接口**：模拟输出

### 3. 设备控制执行器

#### 3.1 继电器模块
- **型号**：2路继电器模块
- **参数**：
  - 控制电压：5V
  - 负载电压：250VAC/30VDC
  - 负载电流：10A
  - 隔离：光耦隔离
- **接口**：数字输出

#### 3.2 RGB LED模块
- **型号**：WS2812B
- **参数**：
  - 数量：5颗
  - 控制方式：单线数字控制
  - 颜色：全彩RGB
- **接口**：数字输出

#### 3.3 红外遥控模块
- **红外发射模块**：
  - 型号：红外发射管+驱动电路
  - 载波频率：38kHz
  - 发射距离：> 5米
  - 接口：数字输出

- **红外接收模块**：
  - 型号：VS1838B
  - 接收频率：38kHz
  - 接收距离：> 5米
  - 接口：数字输出

### 4. 状态指示模块

#### 4.1 RGB LED（状态指示）
- **型号**：WS2812B
- **参数**：
  - 数量：1颗
  - 控制方式：单线数字控制
  - 颜色：全彩RGB
- **接口**：数字输出

### 5. 连接材料
- **面包板**：400孔，带电源总线
- **杜邦线套装**：公对公、公对母、母对母，40根/套
- **排针排母**：2.54mm间距，40P

### 6. 外壳与结构
- **塑料外壳**：透明塑料盒，150×100×50mm
- **固定螺丝**：M3×10mm，4颗

## 🎯 功能需求

### 1. 分控硬件特性
- **无用户交互界面**：分控1没有物理按键、显示屏等交互界面
- **LED状态指示**：通过RGB LED颜色指示工作状态，不提供其他交互
- **蓝牙配网功能**：初始使用蓝牙BLE进行WiFi配网
- **Web配置界面**：连接WiFi后提供Web界面配置控制设备
- **USB串口调试**：通过USB接口进行调试和配置

### 2. 配网功能

#### 2.1 配网方式支持
分控支持两种配网方式：
1. **USB串口配网**（推荐）：通过USB接口连接电脑，使用串口终端进行配网
2. **蓝牙BLE配网**（可选）：通过手机App进行配网

#### 2.2 USB串口配网流程（推荐）
- **功能描述**：通过USB接口连接电脑，使用串口终端进行WiFi配网
- **硬件要求**：ESP32内置USB转串口功能，无需额外模块
- **配网流程**：
  1. 使用USB线连接分控到电脑
  2. 打开串口终端（如Arduino IDE串口监视器、PuTTY）
  3. 配置串口参数：波特率115200，8位数据位，无校验位，1位停止位
  4. 分控启动时检测WiFi配置，如无配置，进入串口配网模式
  5. 串口终端显示配网提示信息
  6. 用户输入WiFi SSID和密码
  7. 分控尝试连接WiFi，显示连接进度
  8. 连接成功后，RGB LED显示绿色
  9. 连接失败后，RGB LED显示红色，重新进入配网模式

#### 2.3 蓝牙BLE配网流程（可选）
- **功能描述**：通过手机App进行WiFi配网
- **配网流程**：
  1. 分控启动时检测WiFi配置
  2. 如果无WiFi配置，进入蓝牙配网模式
  3. 分控启动蓝牙BLE，名称格式：`SmartHome_Slave1_xxxx`
  4. 手机App搜索并连接蓝牙设备
  5. 用户在手机App输入WiFi SSID和密码
  6. 分控接收WiFi信息，断开蓝牙
  7. 分控尝试连接WiFi
  8. 连接成功后，RGB LED显示绿色
  9. 连接失败后，RGB LED显示红色，重新进入配网模式

### 3. Web配置界面

#### 3.1 Web服务器功能
- **功能描述**：分控连接WiFi后提供Web配置界面
- **访问方式**：手机App扫描发现分控IP，或通过内网IP直接访问
- **访问地址**：`http://分控IP`（如：http://192.168.1.101）
- **Web界面功能**：
  - 扫描并添加控制设备
  - 设备状态监控
  - 设备控制操作
  - 红外遥控学习
  - 系统信息查看

#### 3.2 设备添加流程
1. 手机App扫描内网，发现分控设备
2. 点击"添加设备"，进入Web配置界面
3. 点击"扫描设备"按钮
4. 分控扫描已连接的控制设备
5. 显示设备列表（继电器、RGB LED、红外发射）
6. 用户选择要添加的设备
7. 设置设备名称和位置
8. 点击"保存配置"
9. 设备添加成功，可在主控中控制

### 4. 红外遥控学习功能

#### 4.1 红外信号采集
- **功能描述**：学习红外遥控器的按键信号
- **学习流程**：
  1. 在Web界面点击"红外学习"
  2. 选择要学习的设备类型（空调/电视/音响等）
  3. 选择要学习的按键（电源、温度+、温度-等）
  4. 系统进入学习模式，RGB LED显示蓝色
  5. 将遥控器对准红外接收头
  6. 按下遥控器按键
  7. 分控采集红外信号
  8. 采集成功，RGB LED显示绿色
  9. 保存信号到EEPROM

#### 4.2 红外信号发射
- **功能描述**：发射已学习的红外信号控制设备
- **发射流程**：
  1. 主控发送控制指令（如"打开空调"）
  2. 分控接收指令
  3. 从EEPROM读取对应的红外信号
  4. 通过红外发射管发射信号
  5. 设备响应控制

#### 4.3 红外信号存储
- **存储方式**：EEPROM
- **存储容量**：最多存储10个遥控码
- **存储结构**：
  - 设备名称（空调/电视/音响）
  - 按键名称（电源/温度+/温度-/模式）
  - 红外信号数据（原始PWM数据）
  - 信号长度
  - 载波频率（38kHz）

#### 4.4 支持的设备类型
- **空调**：格力、美的、海尔等主流品牌
- **电视**：小米、海信、TCL等主流品牌
- **音响**：索尼、JBL、漫步者等主流品牌
- **机顶盒**：电信、移动、联通等主流品牌
- **风扇**：艾美特、美的、格力等主流品牌

### 5. 环境监测功能

#### 5.1 温湿度监测
- **功能描述**：实时监测环境温度和湿度
- **监测频率**：1-10秒
- **数据上报**：通过MQTT上报主控和云端
- **异常报警**：温度>35℃或<5℃时报警

#### 5.2 空气质量监测
- **功能描述**：实时监测空气质量（CO、酒精、苯等）
- **监测频率**：1-10秒
- **数据上报**：通过MQTT上报主控和云端
- **异常报警**：空气质量>300ppm时报警

#### 5.3 光照强度监测
- **功能描述**：实时监测环境光照强度
- **监测频率**：1-10秒
- **数据上报**：通过MQTT上报主控和云端
- **应用场景**：自动调节灯光亮度

### 6. 设备控制功能

#### 6.1 继电器控制
- **功能描述**：控制继电器开关，控制电器设备
- **控制方式**：MQTT指令 / Web界面
- **控制内容**：
  - 继电器1：控制插座1（如台灯）
  - 继电器2：控制插座2（如风扇）
- **反馈机制**：执行结果反馈

#### 6.2 RGB LED控制
- **功能描述**：控制RGB LED灯的颜色和亮度
- **控制方式**：MQTT指令 / Web界面
- **控制内容**：
  - 颜色设置（红、绿、蓝）
  - 亮度调节（0-255）
  - 效果模式（常亮、呼吸、闪烁）
- **反馈机制**：执行结果反馈

#### 6.3 场景模式
- **功能描述**：预设场景模式，一键执行
- **场景模式**：
  - **回家模式**：打开灯光，调节到舒适亮度
  - **离家模式**：关闭所有灯光和电器
  - **睡眠模式**：关闭电器，灯光调暗
  - **阅读模式**：灯光调亮，适合阅读

### 7. LED状态指示功能

#### 7.1 RGB LED颜色定义
- **功能描述**：通过RGB LED颜色指示分控工作状态
- **颜色定义**：
  - **红色**：配网模式（等待WiFi配置）
  - **绿色**：正常运行（WiFi连接成功，设备在线）
  - **蓝色**：红外学习模式（正在学习红外信号）
  - **黄色**：离线模式（WiFi连接断开）
  - **紫色**：系统错误或异常
  - **白色**：设备控制中（正在执行控制指令）

#### 7.2 LED闪烁模式
- **常亮**：正常运行状态
- **呼吸灯**：配网模式（每秒一次呼吸）
- **快速闪烁**：错误状态（2Hz）
- **慢速闪烁**：离线模式（1Hz）

### 8. USB串口调试功能

#### 8.1 调试接口说明
- **功能描述**：通过USB接口连接电脑，使用串口终端进行调试和配置
- **硬件支持**：ESP32内置USB转串口功能，无需额外模块
- **接口类型**：Micro USB或Type-C接口
- **传输速率**：最高115200 bps
