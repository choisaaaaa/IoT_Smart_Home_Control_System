# 智慧酒店物联网控制系统 - 三层架构楼控需求

## 📋 文档概述

### 文档说明
本文档描述智慧酒店物联网控制系统的三层架构中楼控硬件需求，包括硬件组成、功能需求、接口定义、调试功能等。

### 设备定位
楼控是智慧酒店三层架构的环境监测与服务支持设备，负责环境监测、数据采集、数据上报、状态指示、辅助控制等功能。

---

## 🔧 硬件组成

### 1. 核心控制模块

#### 1.1 ESP32主控板
- **型号**：ESP32-WROOM-32
- **参数**：
  - CPU：双核Xtensa LX6，240MHz
  - Flash：4MB
  - RAM：520KB
  - 无线：WiFi 802.11 b/g/n
  - 接口：GPIO、UART、I2C、SPI、ADC、DAC
- **功能**：核心控制、通信管理

#### 1.2 电源模块
- **参数**：DC 5V/1A
- **接口**：Micro USB
- **功能**：为终端供电

### 2. 环境监测模块

#### 2.1 温湿度传感器
- **型号**：DHT11
- **参数**：
  - 工作电压：3.3V-5V
  - 温度范围：0-50℃，误差±2℃
  - 湿度范围：20-90%RH，误差±5%RH
- **接口**：GPIO
- **功能**：环境温湿度监测

#### 2.2 空气质量传感器
- **型号**：MQ-135
- **参数**：
  - 工作电压：5V
  - 检测气体：氨气、硫化物、烟雾
  - 检测范围：10-1000ppm
- **接口**：ADC
- **功能**：空气质量监测

#### 2.3 光照传感器
- **型号**：光敏电阻模块
- **参数**：
  - 工作电压：3.3V-5V
  - 检测范围：0-1000lux
- **接口**：ADC
- **功能**：光照强度监测

### 3. 辅助控制模块

#### 3.1 继电器模块
- **型号**：2路继电器模块
- **参数**：
  - 控制电压：5V
  - 负载电压：250VAC/30VDC
  - 负载电流：10A
- **接口**：GPIO
- **功能**：辅助设备控制（灯光、插座）

### 4. 状态指示模块

#### 4.1 RGB LED模块
- **型号**：WS2812B
- **参数**：
  - 数量：5颗
  - 控制方式：单线数字控制
  - 颜色：全彩RGB
- **接口**：GPIO
- **功能**：工作状态指示

### 5. 连接材料
- **面包板**：400孔，带电源总线
- **杜邦线套装**：公对公、公对母、母对母，40根/套
- **排针排母**：2.54mm间距，40P
- **电源线**：DC 5.5×2.1mm接口

### 6. 外壳与结构
- **塑料外壳**：透明塑料盒，120×80×40mm
- **固定螺丝**：M3×10mm，4颗

## 🎯 功能需求

### 1. 环境监测功能

#### 1.1 功能描述
实时监测环境参数

#### 1.2 功能要求
- 温湿度监测（DHT11）
- 空气质量监测（MQ-135）
- 光照强度监测（光敏电阻）
- 数据实时上报

### 2. 数据采集功能

#### 2.1 功能描述
实时采集环境数据

#### 2.2 功能要求
- 采集频率：1-10秒
- 数据准确性：误差 < 5%
- 数据完整性：无丢失

### 3. 数据上报功能

#### 3.1 功能描述
将环境数据上报云端

#### 3.2 功能要求
- 上报协议：MQTT
- 上报频率：1-10秒
- 数据格式：JSON
- 断线重连：自动重连

### 4. 状态指示功能

#### 4.1 功能描述
RGB LED显示工作状态

#### 4.2 功能要求
- 正常工作：绿色LED
- 通信异常：红色LED
- 数据采集中：蓝色LED

### 5. 辅助控制功能

#### 5.1 功能描述
支持灯光、插座等基础设备控制

#### 5.2 功能要求
- 2路继电器控制
- 本地控制（断网可用）
- 远程控制
- 定时控制

## 🔌 硬件接口定义

### 主控板引脚定义

| ESP32引脚 | 功能 | 连接设备 | 说明 |
|-----------|------|----------|------|
| 3.3V | 电源输出 | 所有3.3V设备 | 供电 |
| GND | 接地 | 所有设备接地 | 接地 |
| 5V | 电源输入 | 5V电源模块 | 输入5V电源 |
| GPIO15 | GPIO | 温湿度传感器 | DHT11数据线 |
| GPIO2 | GPIO | 空气质量传感器 | MQ-135模拟输入 |
| GPIO34 | GPIO | 光照传感器 | 光敏电阻模拟输入 |
| GPIO13 | GPIO | 继电器1 | 辅助设备控制1 |
| GPIO14 | GPIO | 继电器2 | 辅助设备控制2 |
| GPIO5 | GPIO | RGB LED | WS2812B数据线 |
| GPIO13 | UART TX | USB转TTL | 调试串口 |
| GPIO3 | UART RX | USB转TTL | 调试串口 |

## 🔧 调试功能

### 1. USB串口调试
- **功能描述**：通过USB串口进行调试
- **功能要求**：
  - 串口波特率：115200
  - 支持日志输出
  - 支持命令调试
  - 支持固件升级

### 2. WiFi配网
- **功能描述**：通过WiFi进行配网
- **功能要求**：
  - 支持热点模式配网
  - 支持STA模式配网
  - 支持配网状态显示
  - 支持配网日志记录
