# 物联网智能家居控制系统 - 分控2硬件设计文档

## 📋 文档概述

### 文档说明
本文档详细描述分控2（安防管理终端）的硬件设计，包括硬件组成、接线表、原理图、功能说明等内容。

### 适用范围
- 硬件开发
- 硬件调试
- 硬件维护
- 硬件培训

### 文档版本
- **v1.0.0** - 2024年1月
  - 初始版本

## 📦 硬件组成

### 分控2硬件清单

| 序号 | 组件名称 | 型号/规格 | 数量 | 说明 |
|------|----------|-----------|------|------|
| 1 | 主控板 | ESP32-WROOM-32 | 1 | 核心控制器，双核240MHz，WiFi+蓝牙 |
| 2 | RFID读卡器 | MFRC522 | 1 | RFID读卡，门禁控制 |
| 3 | 门磁传感器 | 干簧管 | 1 | 门窗状态监测 |
| 4 | 人体传感器 | HC-SR501 | 1 | 人体检测，入侵报警 |
| 5 | 蜂鸣器 | 3V 蜂鸣器 | 1 | 报警提示 |
| 6 | 电控锁 | 12V电磁锁 | 1 | 门禁控制 |
| 7 | 继电器模块 | 5V双路继电器 | 1 | 电控锁控制 |
| 8 | 电源模块 | 12V/2A | 1 | 供电，适配器接口 |
| 9 | RGB LED | WS2812B | 1 | 状态指示 |
| 10 | 电阻/电容 | 若干 | 若干 | 电路配套元件 |
| 11 | PCB板 | 定制PCB | 1 | 硬件载体 |

| GPIO13 | 继电器1 | 输出 | 电控锁控制 |
| GPIO14 | 人体传感器 | 输入 | 人体检测输入 |
| GPIO15 | 门磁传感器 | 输入 | 门窗状态检测输入 |
| GPIO16 | UART RX | 输入 | 调试串口RX |
| GPIO17 | UART TX | 输出 | 调试串口TX |
| GPIO18 | RFID RST | 输出 | SPI复位 |
| GPIO19 | RFID MISO | 输入 | SPI主入从出 |
| GPIO21 | RFID MOSI | 输出 | SPI主出从入 |
| GPIO23 | RFID SCK | 输出 | SPI时钟 |
| GPIO27 | RFID IRQ | 输入 | SPI中断（可选） |

## 🔍 硬件调试

### 调试接口

#### 1. USB串口调试
- **接口**：USB Type-C
- **芯片**：CH340或CP2102
- **波特率**：115200
- **引脚**：GPIO17(TX)、GPIO16(RX)

#### 2. SPI调试
- **接口**：RFID模块接口
- **引脚**：GPIO5(SS)、GPIO18(RST)、GPIO23(SCK)、GPIO19(MISO)、GPIO21(MOSI)
- **频率**：1MHz

#### 3. 传感器调试
- **门磁传感器**：GPIO15，数字输入
- **人体传感器**：GPIO14，数字输入

#### 1. 电源调试

1. 连接USB转TTL模块

3. 配置波特率115200
4. 查看启动日志

#### 3. RFID调试
1. 连接RFID模块

#### 4. 传感器调试
1. 连接门磁传感器
2. 连接人体传感器
3. 运行传感器测试程序
4. 检查传感器状态

#### 5. 蜂鸣器调试
1. 连接蜂鸣器
2. 运行蜂鸣器测试程序
3. 检查蜂鸣器发声

#### 6. 继电器调试
1. 连接继电器模块
2. 运行继电器测试程序
3. 检查继电器开关

#### 7. LED调试
1. 连接RGB LED
2. 运行LED测试程序
3. 检查LED颜色显示

## 🛠️ 硬件维护

### 常见问题

#### 1. 电源问题
**现象**：设备无法启动
**原因**：
- 电源电压不足
- 电源连接不良
- 电源模块损坏

**解决**：
- 检查电源电压
- 检查电源连接
- 更换电源模块

#### 2. 串口问题
**现象**：串口无输出
**原因**：
- 串口连接错误
- 波特率设置错误
- 串口芯片损坏

**解决**：
- 检查串口连接
- 检查波特率设置
- 更换串口芯片

#### 3. RFID问题
**现象**：RFID读卡失败
**原因**：
- RFID连接错误
- RFID模块损坏
- 卡片问题

**解决**：
- 检查RFID连接
- 更换RFID模块
- 更换卡片

#### 4. 传感器问题
**现象**：传感器数据异常
**原因**：
- 传感器连接错误
- 传感器损坏
- 程序问题

**解决**：
- 检查传感器连接
- 更换传感器
- 检查程序

#### 5. 蜂鸣器问题
**现象**：蜂鸣器不发声
**原因**：
- 蜂鸣器连接错误
- 蜂鸣器损坏
- 程序问题

**解决**：
- 检查蜂鸣器连接
- 更换蜂鸣器
- 检查程序

#### 6. 继电器问题
**现象**：继电器不动作
**原因**：
- 继电器连接错误
- 继电器损坏
- 程序问题

**解决**：
- 检查继电器连接
- 更换继电器
- 检查程序

#### 7. 电控锁问题
**现象**：电控锁不动作
**原因**：
- 电控锁连接错误
- 电控锁损坏
