# 物联网智能家居控制系统 - 主控中心硬件设计文档

## 📋 文档概述

### 文档说明
本文档详细描述主控中心的硬件设计，包括硬件组成、接线表、原理图、功能说明等内容。

### 适用范围
- 硬件开发
- 硬件调试
- 硬件维护
- 硬件培训

### 文档版本
- **v1.0.0** - 2024年1月
  - 初始版本

## 📦 硬件组成

### 主控中心硬件清单

| 序号 | 组件名称 | 型号/规格 | 数量 | 说明 |
|------|----------|-----------|------|------|
| 1 | 主控板 | ESP32-WROOM-32 | 1 | 核心控制器，双核240MHz，WiFi+蓝牙 |
| 2 | 扬声器 | 3W 8Ω | 1 | 音频输出，语音反馈 |
| 3 | 功放芯片 | PAM8403 | 1 | 音频放大，驱动扬声器 |
| 4 | OLED显示屏 | 0.96寸I2C | 1 | 本地显示，128x64分辨率 |
| 5 | 麦克风 | INMP441 I2S | 1 | 音频输入，语音识别 |
| 6 | 按键模块 | 4键独立按键 | 1 | 语音触发，本地控制 |
| 7 | 蜂鸣器 | 3V 蜂鸣器 | 1 | 声音提示，报警提示 |
| 8 | RGB LED | WS2812B | 1 | 状态指示，颜色提示 |
| 9 | 电源模块 | 5V/2A | 1 | 供电，USB Type-C接口 |
| 10 | 电阻/电容 | 若干 | 若干 | 电路配套元件 |
| 11 | PCB板 | 定制PCB | 1 | 硬件载体 |

| GND | 接地 | 接地 | 系统接地 |

| GPIO4 | 按键4 | 输入 | 按键输入，下拉电阻 |
| GPIO5 | RGB LED | 输出 | WS2812B数据输出 |
| GPIO13 | UART TX | 输出 | 调试串口TX |
| GPIO3 | UART RX | 输入 | 调试串口RX |
| GPIO15 | 按键1 | 输入 | 按键输入，下拉电阻 |
| GPIO16 | 蜂鸣器 | 输出 | PWM控制 |
| GPIO17 | 扬声器R | 输出 | 右声道PWM输出 |
| GPIO18 | 扬声器L | 输出 | 左声道PWM输出 |
| GPIO19 | 功放SHUTDOWN | 输出 | 功放使能控制 |
| GPIO21 | I2C SDA | 双向 | OLED I2C数据线 |
| GPIO22 | I2C SCL | 双向 | OLED I2C时钟线 |
| GPIO25 | I2S BCLK | 输出 | 麦克风位时钟 |
| GPIO26 | I2S LRCLK | 输出 | 麦克风左右声道时钟 |
| GPIO27 | I2S DOUT | 输入 | 麦克风数据输出 |

## 🔍 硬件调试

### 调试接口

#### 1. USB串口调试
- **接口**：USB Type-C
- **芯片**：CH340或CP2102
- **波特率**：115200
- **引脚**：GPIO13(TX)、GPIO3(RX)

#### 2. I2C调试
- **接口**：OLED显示屏接口
- **引脚**：GPIO21(SDA)、GPIO22(SCL)
- **地址**：0x3C

#### 3. I2S调试
- **接口**：麦克风接口
- **引脚**：GPIO25(BCLK)、GPIO26(LRCLK)、GPIO27(DOUT)

### 调试步骤

#### 1. 电源调试
1. 连接5V电源
2. 测量3.3V输出
3. 检查各模块供电

#### 2. 串口调试
1. 连接USB转TTL模块
2. 打开串口调试软件
3. 配置波特率115200
4. 查看启动日志

#### 3. OLED调试
1. 连接OLED显示屏
2. 运行测试程序
3. 检查显示内容

#### 4. 音频调试
1. 连接麦克风和扬声器
2. 运行音频测试程序
3. 检查音频采集和播放

#### 5. 按键调试
1. 连接按键模块
2. 运行按键测试程序
3. 检查按键响应

## 🛠️ 硬件维护

### 常见问题

#### 1. 电源问题
**现象**：设备无法启动
**原因**：
- 电源电压不足
- 电源连接不良
- 电源模块损坏

**解决**：
- 检查电源电压
- 检查电源连接
- 更换电源模块

#### 2. 串口问题
**现象**：串口无输出
**原因**：
- 串口连接错误
- 波特率设置错误
- 串口芯片损坏

**解决**：
- 检查串口连接
- 检查波特率设置
- 更换串口芯片

#### 3. OLED问题
**现象**：OLED不显示
**原因**：
- I2C连接错误
- OLED损坏
- 程序问题

**解决**：
- 检查I2C连接
- 更换OLED
- 检查程序

#### 4. 音频问题
**现象**：音频无输出
**原因**：
- 麦克风连接错误
- 功放损坏
- 扬声器损坏

**解决**：
- 检查音频连接
- 更换功放
- 更换扬声器

#### 5. 按键问题
**现象**：按键无响应
**原因**：
- 按键连接错误
- 按键损坏
- 程序问题

**解决**：
- 检查按键连接
- 更换按键
- 检查程序
