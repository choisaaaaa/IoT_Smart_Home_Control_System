## 🛠️ 硬件调试

### USB串口调试

**调试工具**：
- USB转TTL模块
- 串口调试软件（如XCOM、SecureCRT）

**调试步骤**：
1. 连接USB转TTL模块到ESP32
2. 安装USB转TTL驱动
3. 打开串口调试软件
4. 配置串口参数（115200 8N1）
5. 上传固件并查看调试信息

### 蓝牙配网

**配网流程**：
1. 设备启动进入配网模式
2. 手机APP扫描设备蓝牙
3. 输入WiFi信息
4. 设备连接WiFi
5. 配网完成
