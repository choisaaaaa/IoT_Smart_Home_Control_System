# IoT Smart Home Control System

## Overview
The IoT Smart Home Control System is an innovative solution designed to automate various aspects of home management using connected devices. This project aims to enhance energy efficiency, improve home security, and provide convenience to users through a centralized control system.

## Features
- Remote control of home appliances
- Real-time monitoring of home environment
- Integration with voice assistants
- Secure access and control features

## Technologies Used
- **Programming Languages:** Python, JavaScript
- **Frameworks:** Flask, React
- **Databases:** MongoDB
- **Protocols:** MQTT, HTTP
- **Cloud Services:** AWS, Azure

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/choisaaaaa/IoT_Smart_Home_Control_System.git
   cd IoT_Smart_Home_Control_System
   ```
2. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up the database:
   Follow the instructions in the `database_setup` directory.

## Usage
1. Start the server:
   ```bash
   python app.py
   ```
2. Access the dashboard by navigating to `http://localhost:5000` in your web browser.
3. Configure your devices through the dashboard.

## Architecture
The architecture of the IoT Smart Home Control System involves:
- **Frontend:** User interface built using React
- **Backend:** REST API developed with Flask
- **Database:** MongoDB for storing user and device data
- **Communication:** MQTT protocol for real-time communication between devices
